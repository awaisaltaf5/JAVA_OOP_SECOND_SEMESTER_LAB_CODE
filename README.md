

# Java OOP Second Semester Lab Code

This repository contains Java Object-Oriented Programming (OOP) lab exercises from my second semester.

## Topics Covered
- Classes and Objects
- Inheritance
- Polymorphism
- Encapsulation
- Abstraction
- Interfaces
- Exception Handling
- File Handling

## How to Use
1. Clone the repository:
   ```bash
   git clone https://github.com/awaisaltaf5/JAVA_OOP_SECOND_SEMESTER_LAB_CODE.git
   ```
2. Open the project in your preferred IDE (IntelliJ, Eclipse, etc.).
3. Explore the code examples for understanding core OOP concepts.

## Purpose
This repository serves as a collection of practice code for understanding and applying OOP principles in Java.
